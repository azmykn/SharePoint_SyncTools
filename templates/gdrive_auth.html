Google Drive sign-in

Because of Google’s security model, the login helper must run as a separate process

1
Open a terminal in the project root (same folder as gdrive_login.py).
2
Run (click to copy). Use your venv’s Python if you use one:
python gdrive_login.py
Windows: .venv\Scripts\python.exe gdrive_login.py
Linux/macOS: .venv/bin/python gdrive_login.py
3
A browser window opens — complete Google sign-in and allow access.
4
When it succeeds, return here and open the dashboard again.
Done — back to dashboard

After the first successful login you usually do not need to repeat this.
